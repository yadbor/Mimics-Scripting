mimics.experimental module¶

mimics.experimental.create_transformation_from_object(object)¶

Creates transformation object.

Parameters

object (mimics.Object) – Object from which the transformation is created.

Returns

Transformation object

Return type

mimics.Transformation

Exceptions

ValueError (reason(s): [‘Transformation object can not be created if tranformation matrix is not rigid.’])

Example

part_a = mimics.data.parts[0]

device_transformation = mimics.experimental.create_transformation_from_object(part_a)
mimics.experimental.invert_transformation(transformation)¶

Inverts input transformation object.

Parameters

transformation (mimics.Transformation) – Transformation object.

Example

t = mimics.data.transformations[0]
# the same as t.invert()
mimics.experimental.invert_transformation(t)
mimics.experimental.set_transformation(object, transformation)¶

Sets (replaces) transformation to object.

Parameters

  • object (Union[mimics.Part, mimics.Osteotomy, mimics.analyze.Point, mimics.analyze.Line, mimics.analyze.Circle, mimics.analyze.Sphere, mimics.analyze.Cylinder, mimics.analyze.Spline, mimics.analyze.Plane, mimics.analyze.Centerline]) – Object for which to set transformation.

  • transformation (mimics.Transformation) – Transformation object.

Example

part_a = mimics.data.parts[0]
part_b = mimics.data.parts[1]

transformation_a = mimics.experimental.create_transformation_from_object(part_a)

p = mimics.experimental.set_transformation(part_b, transformation_a)
