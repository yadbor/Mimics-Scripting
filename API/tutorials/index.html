Tutorials¶

Note: You can find the Python files (.py) of the tutorials in the installation directory of Materialise software, inside the Mimics installation folder: \..\Help\Tutorial Scripts. Additionaly, a copy of the scripts is created in C:\MedData\Scripts. In that location you can also save your Python scripts. This is the default path of the Scripting library. See in Section 2.2 how to modify Mimics preferences for the Scripting library.
